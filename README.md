# Global Market Insights - Minimal Repo Skeleton

This archive contains a minimal frontend (Next.js + Tailwind + Recharts) and backend (Express + Socket.io) skeleton
for a Global Market Insights dashboard. Follow the run instructions in each folder.

## Quick start

- Frontend:
  - cd frontend
  - npm install
  - npm run dev

- Backend:
  - cd backend
  - npm install
  - npm run dev

Set NEXT_PUBLIC_WS to your backend URL if needed (default: http://localhost:4000)
